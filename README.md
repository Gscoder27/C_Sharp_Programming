## 🛠️ How to Run

1. **Clone the repository:**
	```bash
	git clone https://github.com/Gscoder27/C_Sharp_Programming.git
	cd C_Sharp_Programming
	```
2. **Open in Visual Studio or VS Code.**
3. **Build and run individual files or projects:**
	- Use the built-in terminal or IDE features to compile and execute `.cs` files.
	- For project files (`.csproj`), use:
	  ```bash
	  dotnet run --project <ProjectName>.csproj
	  ```
